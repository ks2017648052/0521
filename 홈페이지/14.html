<<디렉토리 만들기>>


-새로운 디렉토리를 만드는 명령 : mkdir (make directory)

ex) mkdir aa //aa라는 디렉토리 만듦
    mkdir bb cc dd //bb,cc,dd 디렉토리 만듦

윈도우에서는 새로운 디렉토리(파일)을 만들려면 일일히 폴더를 다 만들어야 하지만
리눅스 환경에서는 한꺼번에 만들 수 있음

-rmdir : 디렉토리를 삭제하는 명령

ex) rmdir aa
    rmdir bb cc dd

-wildcard의 종류에는 여러가지가 있음
 * = all(모든)
 ? = 글자한개

ex) 글자 한개를 지우는 명령 rmdir ?

-rmdir 명령시 디렉토리 안에 내용이 들어있으면 명령이 실행되지 않음, 그러므로 명령을 실행
시키기 전에 디렉토리 안의 내용 먼저 rmdir을 사용하여 비우고 실행시켜야함

-파일 복사 : cp

ex) cp exampke.desktop kk //kk에 example 파일을 복사하시오

-파일 삭제 : rm

ex) rm kk //kk를 삭제하시오

-파일 이동 : mv

ex) mv kk pp //kk를 pp로 이동하시오 (rename의 역할)

-희재 디렉토리에 있는 파일을 내 현재위치에 가져오고 싶을 때에는 cp ~hjyang/Test.java .
.(온점)이 내 현재위치를 의미

-cp ~hhjyang/Test* aaa : 희재 디렉토리에 있는 Test라는 이름을 가진 모든 파일을 aaa로 복사

<<옵션>>

- cp -i : interactive(상호대화) , overwrite가 되지 않도록 물어봄 
- cp -r : recursive(반복) , 디렉토리 안의 모든 내용들을 가져올 수 있도록 하는 명령, 
     반복해서 복사

윈도우에서는 폴더안의 모든 내용을 가져오고 싶으면 ctrl+c,ctrl+v,
리눅스에서는 cp -r

- rm -i
- rm -r

- mv -i
- mv -r


-i,-r의 원리는 비슷함
