*로그아웃하면  alias로 지은 별명들이 다 사라짐, alias는 하나만 해당
*개인이 하는 환경설정을 보관해두는 파일 : .bashrc
*키보드로부터 받아내야되는 내용을 파일로 받아내는 것 : redirection 
*pipeline : 표준 출력이 다른 명령의 표준입력으로 들어가게 하는 것
*wc : word counts ,단어를 세는 명령, 기억하셈
ex> cat > ll.txt
    (내용)
     wc ll.txt
하면 몇 줄인지, 몇 단어인지, 글자가 몇개인지(공백포함) 세어줌

*pipe는 여러단계 가능, 꼭 한 단계만 있을 필요 ㄴㄴ
*wc more /etc/passwd : 리눅스 계정 사용자가 몇명인지 알 수 있음
*grep : 어떤 패턴이 있는 라인을 찾아낸 뒤에 그 패턴을 출력 , 중요하니 꼭 기억 , 특정 패턴을 가지고 있는가 
ex >
grep kim /etc/passwd : /etc/passwd에서 kim이 들어간 내용을 찾아라, 대소문자 구별됨 (대소문자 구별을 안하려면 kim 앞에 i 추가)

ex2>
ls /bin | grep zip : /bin에서 zip이 들어있는 글자 출력

* head /etc/passwd : /etc/passwd의 제일 앞쪽 10줄만 출력
* tail /etc/passwd : /etc/passwd의 제일 뒷쪽 10줄만 출력

*ls /bin | head(or tail) : bin의 앞(or 뒤) 부분 10개만 보임
*head, tail 의 기본은 10개이지만 바꿀수도 있음
ex>
ls /bin | head -n 5 :  5줄만 출력
